1.  What is the difference between var, let, and const?
    Answer:
    The main difference between var, let, and const is in scope and reassignment.

          var is function-scoped and can be re-declared and updated. It’s mostly avoided in modern JavaScript.
          let is block-scoped and can be updated, but cannot be re-declared in the same scope.
          const is also block-scoped, but it cannot be re-assigned or re-declared.

2.  What is the spread operator (...)?
    Answer:
    The spread operator (...) is used to expand or copy elements from arrays or objects, making it easier to merge data or create new copies without modifying the original.

3.  What is the difference between map(), filter(), and forEach()?
    Answer:
    map() creates a new array by transforming each element, filter() creates a new array with elements that match a condition, and forEach() just loops through the array without returning anything.

4.  What is an arrow function?
    Answer:
    An arrow function is a shorter syntax for writing functions in JavaScript, and it does not have its own this (it uses the surrounding context’s this).

5.  What are template literals?
    Answer:
    Template literals are a way to write strings using backticks (`), allowing you to easily include variables and expressions using ${} and write multi-line strings.
